DEFB/DB

[ 前ページへ | 次ページへ ]
機能
1バイト単位でデータをメモリに配置します。
　
書式 
	DEFB	<式>[,式] ... [,式]

または

	DB	<式>[,式] ... [,式]
　
説明

<式>には、任意の定数、文字、または文字列を与えることができます。<式>の内容は、先頭のデータから順にメモリに配置されます。複数の<式>を与える時は、それぞれの<式>をカンマ（,）で区切ってください。

なお、DEFB疑似命令とDB疑似命令はスペルのみ異なり、動作は同じです。
　
参照
DEFH/DH、DEFW/DW、DEFD/DD、DEFL/DL
　
入力例 
	defb	$10		; 定数
	defb	'a'		; 文字
	defb	"TEST"		; 文字列
