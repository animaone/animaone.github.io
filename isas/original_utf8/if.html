IF/IFN

[ 前ページへ | 次ページへ ]
機能
アセンブル条件が真の時評価します。
　
書式 
	IF	<式>

または

	IFN	<式>
　
説明

アセンブル条件の<式>が真の時、つまり<式>の値が0以外の時、次のELSE疑似命令またはENDIF疑似命令までのブロックをアセンブルします。

なお、IF疑似命令とIFN疑似命令はスペルのみ異なり、動作は同じです。
　
参照
IFE、IFB、IFNB、IFDEF、IFNDEF、ELSE、ENDIF
　
入力例 
	if	1		; アセンブル条件が真であるためアセンブ
	lda	#$00		; ルされる
	endif
	if	0		; アセンブル条件が偽であるためアセンブ
	sta	#$00		; ルされない
	endif
